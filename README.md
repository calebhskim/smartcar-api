# smartcar-api
Smartcar API Project

# Install
```js
git clone https://github.com/calebhskim/smartcar-api.git
cd smartcar-api
npm install
```

# Usage
To start the API locally
```js
npm run start
```

To run the tests
```js
npm test
```
